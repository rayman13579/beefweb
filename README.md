# beefweb
Stripped back version of original [beefweb](https://github.com/hyperblast/beefweb)
Web interface plugin for [foobar2000](https://foobar2000.org/) music player.

[![License](https://img.shields.io/github/license/hyperblast/beefweb.svg)](LICENSE)

### Features
- Real-time UI updates
- REST-like API to get current track information


### How to install

#### foobar2000
Open `foo_beefweb-*.fb2k-component` with foobar2000, you will be asked to install the plugin.

### How to use
Point your browser to `http://localhost:8880/foobar2000` to get updates on the current track.
